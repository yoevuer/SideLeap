# SideLeap 资源索引

## 当前状态

本页仅记录当前仍在使用的资源与配置入口，不再引用已删除的 `ktx/`、`utils/`、`entity/` 旧路径。

## 资源清单

| 路径 | 用途 |
|---|---|
| `app/src/main/AndroidManifest.xml` | 应用入口、Service、Provider、权限 |
| `app/src/main/res/values/strings.xml` | 文案资源 |
| `app/src/main/res/values/colors.xml` | 颜色资源 |
| `app/src/main/res/values/themes.xml` | 主题资源 |
| `app/src/main/res/xml/accessibility_service_config.xml` | 无障碍服务配置 |
| `app/src/main/res/xml/app_file_paths.xml` | FileProvider 路径配置 |
| `app/src/main/res/xml/backup_rules.xml` | 备份规则 |
| `app/src/main/res/xml/data_extraction_rules.xml` | 数据提取规则 |
| `app/src/main/res/drawable-nodpi/sideleap.webp` | 应用图标 |
| `app/src/main/res/drawable-xxhdpi/github.png` | 图标资源 |

## 构建与配置

| 路径 | 用途 |
|---|---|
| `app/build.gradle.kts` | 应用构建配置 |
| `app/proguard-rules.pro` | 混淆规则 |
| `app/lint.xml` | Lint 规则 |
| `gradle/libs.versions.toml` | 依赖版本目录 |
| `gradle.properties` | Gradle 属性 |
| `settings.gradle.kts` | Gradle 设置 |

## 说明

- 资源索引只记录当前稳定文件。
- 旧源码路径只应出现在 archived 索引页中。
