# SideLeap 代码索引

## 当前领域

| 领域 | 说明 |
|---|---|
| `ui` | 页面、组件、导航、UI 状态 |
| `gesture` | 手势模型、输入识别、手势语义 |
| `action` | 动作定义、展示、payload、执行分发 |
| `settings` | 配置模型、DataStore、默认值、备份恢复 |
| `launcher` | 应用/Shortcut/图标/启动能力 |
| `freeze` | 冻结业务、名单策略、Shizuku 冻结链路 |
| `overlay` | 快捷启动器浮窗 |
| `system` | Android 底层能力封装 |
| `core` | 跨领域基础设施与应用入口类 |
