# SideLeap

[English](README_EN.md)

SideLeap 是一个面向 Android 的侧边手势、快捷启动与系统动作增强工具。通过可配置的触钮，将返回、主页、音量、应用启动、虚拟鼠标等操作绑定到滑动、长滑、单击和长按手势上。

当前版本：`v1.6.0`

## 功能特性

- **手势触发** — 左侧、右侧、底部三条触发区域；短滑、长滑、单击、长按四种手势；每个触钮独立配置手势角度。
- **子手势** — 每个触钮可配置 8 个方向的子手势动作，触发后在全屏任意位置滑动执行。
- **动作面板** — 长滑即时触发时可展开多动作面板，支持弧形和网格两种布局。
- **快捷启动** — 启动应用与快捷方式、隐藏应用、最近/频率记录。
- **系统动作** — 返回、主页、最近任务、通知面板、快捷面板、音量调节、静音、媒体控制等。
- **工具动作** — 密码生成器、模拟点击、虚拟鼠标（含连续模式）、滚动屏幕等。
- **小窗启动** — 应用小窗位置、边缘留白和位置补偿配置。
- **冻结管理** — 配合 Shizuku 冻结/解冻应用、一键冻结、保护名单。
- **个性化** — 动态配色、深色模式、可自定义颜色的动画轨迹、图标缩放与背景色。

## 系统要求

- Android 13 及以上（API 33+）
- 需要启用无障碍服务以执行手势动作
- 冻结/解冻应用需要 Shizuku 环境

## 权限说明

- **无障碍服务**：用于执行返回、主页、最近任务、点击、打开面板等动作。不收集个人数据。
- **查询应用列表**：用于应用选择、快捷方式、黑名单、隐藏应用和冻结管理。
- **Shizuku Provider**：用于冻结/解冻等需更高系统能力的操作。
- **电池优化与自启动**：用于提高后台手势服务存活率。
- 其他权限（通知、相机、蓝牙等）仅在对应用系统动作时按需申请。

## 构建

```bash
# Debug
./gradlew assembleDebug

# Release
./gradlew assembleRelease
```

APK 输出路径：`app/build/outputs/apk/{debug,release}/`
